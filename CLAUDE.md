# npresence

Live user presence, heartbeat tracking, and activity logging.

## Key Classes
- `PresenceService` — track online/offline status
- `PresencePresenterTrait` — mixin for presenters
- DI Extension: `PresenceExtension`

## Conventions
- PHP 8.2+ with `declare(strict_types=1)`
- PSR-4 autoloading in `src/`
- Nette DI Extension for service registration
- PHPStan level 9
- Part of `jansuchanek/*` on Packagist
- GitHub: https://github.com/JanSuchanek/npresence
